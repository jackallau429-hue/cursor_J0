# 🐱 The Daily Meowsletter — YouTube Project

> *"Today's news, judged by a cat."*

Semi-automated YouTube channel using AI tools to produce daily news-commentary
shorts and weekly long-form recap videos, fronted by a real orange tabby cat
acting as a deadpan news anchor.

**Goal:** US$1,000+ / month passive income within 6–9 months.
