# Account Setup Checklist

One-time setup. Do this once, do it right.

---

---

## A. Google / Gmail (~10 min)

1. Go to <https://accounts.google.com/signup>, choose **For my personal use**
2. Username = lowercase channel name
3. Password ≥16 chars, generated by a password manager (1Password / Bitwarden)
4. Phone verify → finish creation
5. **IMMEDIATELY enable 2FA**:
   - <https://myaccount.google.com/security> → **2-Step Verification**
   - Use **Google Authenticator** or **Authy** (NOT SMS-only)
6. Set **Recovery email** + **Recovery phone**
7. Download **Backup codes** → store in password manager + printed copy

---

## B. YouTube channel (~15 min)

1. Sign in to <https://www.youtube.com> with the new Gmail
2. Top-right avatar → **Create a channel**
3. Choose **Use a custom name** (this creates a **Brand Account** — required for business / multi-admin / future sale)
4. Enter channel name (e.g. `The Daily Meowsletter`)
5. **YouTube Studio → Settings → Channel → Basic info**:
   - Country of residence: **United States** (or target market)
   - Default language: **English**
6. **Settings → Channel → Advanced settings**:
   - Audience: **"No, set this channel as not made for kids"**
     (CRITICAL — "Made for kids" disables comments and crashes RPM)
   - Automatic captions: enabled
7. Reserve handle at <https://www.youtube.com/handle> → `@dailymeowsletter` (or chosen)
8. Skip Customization (banner / about) for now — will be written separately
9. **Do NOT apply for monetization yet** — requires 1k subs + 4k watch hours

---

## C. Do NOT do (until ready)

- Do not use a personal Gmail or personal YouTube channel
- Do not flag channel as "Made for kids"
- Do not link AdSense yet (wait until monetization eligibility)
- Do not upload any videos until channel branding (banner, about, trailer) is approved
